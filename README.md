# Tic Tac Toe Game 🎮

A simple **Tic Tac Toe** game built using **HTML**, **CSS**, and **JavaScript**.

## 📌 Features
- 2 Player Game (X vs O)
- Winning detection
- Reset Game option
- Clean and responsive UI
  
## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
🚀 How to Run
- Download or clone the repository.
- Open the index.html file in your browser.
- Enjoy playing!
